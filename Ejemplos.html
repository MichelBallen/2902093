Códigos Básicos de Git con Ejemplos

Comandos con Ejemplos

  • git init mi_proyecto
    Ejemplo: Inicializa un nuevo repositorio local llamado "mi_proyecto".
  • git clone https://github.com/usuario/repositorio.git mi_copia_local
    Ejemplo: Clona un repositorio existente desde GitHub (https://github.com/usuario/repositorio.git) con el nombre "mi_copia_local".
  • git add index.html styles.css script.js
    Ejemplo: Agrega los archivos "index.html", "styles.css" y "script.js" al área de preparación para un commit.
  • git commit -m "Corregir error en la función principal"
    Ejemplo: Guarda los cambios añadidos como un commit con el mensaje "Corregir error en la función principal".
  • git push origin master
    Ejemplo: Envía los cambios locales del branch "master" al repositorio remoto "origin".

Estos son solo algunos ejemplos básicos.
